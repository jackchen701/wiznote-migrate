# 介绍

该工具主要是辅助从为知笔记中迁移笔记到其他 AI Power 工具，为知笔记曾经是一款十分优秀的产品，但是随着 AI 时代的到来目前总体还是比较落后，迁移过程也确实被折腾的有些难过。

# 使用方法
```bash
wiz_export --output '/Users/xx/' --userId 'xx' --password 'xx' --folders '/日记/,/工作/'
```

# 注意事项

目前迁移的数据尽量保持了原来的目录，但是有些新日期的日记是无法导出的，得你自己手动去导出，具体的原因我想大家也能明白，大概是 202403 月之后基本都导不出来了，得从软件导出，软件不提供批量导出的功能。
导出的文件格式为 markdown，markdown 中引用的图片是按照 obsidian 的目录结构组织的，也就是附件会放在 assets 目录下。
